# CodeSnippet Manager

## English

CodeSnippet Manager is a desktop application developed in Java that allows users to store, organize and manage code snippets.

Main features:

* Create, edit and delete snippets
* Manage users, languages, categories and tags
* Search snippets by title
* Store data using Hibernate and MySQL
* Java Swing graphical interface

Technologies used:

* Java 21
* Java Swing
* Hibernate ORM
* JPA
* MySQL
* Maven
* Plantuml

## Español

CodeSnippet Manager es una aplicación de escritorio desarrollada en Java que permite almacenar, organizar y gestionar fragmentos de código.

Características principales:

* Crear, editar y eliminar snippets
* Gestionar usuarios, lenguajes, categorías y etiquetas
* Buscar snippets por título
* Almacenar datos mediante Hibernate y MySQL
* Interfaz gráfica desarrollada con Java Swing

Tecnologías utilizadas:

* Java 21
* Java Swing
* Hibernate ORM
* JPA
* MySQL
* Maven
* Plantuml
